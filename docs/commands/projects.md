# projects

List all projects in your organization.

## Usage

```bash
vaultuner projects
```

## Examples

```bash
vaultuner projects
```

## Output

```
┏━━━━━━━━━━━━┓
┃ Project    ┃
┡━━━━━━━━━━━━┩
│ myapp      │
│ backend    │
│ frontend   │
└────────────┘
```

## Notes

Projects are created automatically when you create secrets. This command lists all projects that exist in your Bitwarden organization.
